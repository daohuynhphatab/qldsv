HƯỚNG DẪN SỬ DỤNG SCRIPT SEED DATA

1. Đảm bảo database 'qldsv' đã được tạo và các bảng đã tồn tại
2. Mở MySQL Workbench hoặc công cụ quản lý MySQL khác
3. Kết nối đến database 'qldsv' với user root và password 12345
4. Mở file seed_data.sql và chạy toàn bộ script

DỮ LIỆU MẪU BAO GỒM:

GIÁO VIÊN:
- gv1 (password: hash mẫu)
- gv2 (password: hash mẫu)
- gv3 (password: hash mẫu)

MÔN HỌC:
- Lập trình web (3 tín chỉ)
- Điện toán đám mây (3 tín chỉ)
- Lập trình quản lí (3 tín chỉ)

LỚP:
- DTH235

SINH VIÊN:
- DTH235001 đến DTH235020 (20 sinh viên)
- Tất cả thuộc lớp DTH235

PHÂN CÔNG GIẢNG DẠY:
- gv1 dạy Lập trình web cho lớp DTH235
- gv2 dạy Điện toán đám mây cho lớp DTH235
- gv3 dạy Lập trình quản lí cho lớp DTH235

LƯU Ý:
- Password được hash bằng BCrypt, bạn có thể thay đổi thành password thật
- Có thể uncomment phần INSERT grades để thêm điểm mẫu
- Đảm bảo chạy script theo thứ tự để tránh lỗi foreign key
